  ## À propos

  **Koi's Story** est une boutique en ligne permettant de consulter et réserver des carpes koï
  de la lignée Konishi. Le visiteur peut parcourir le catalogue, filtrer par variété, taille
  ou prix, puis contacter le vendeur directement via WhatsApp en un clic.

  **Fonctionnalités principales**

  - Catalogue filtrable (variété / taille / prix)
  - Fiche produit avec galerie photos et badge Lignée Konishi
  - Bouton "Commander via WhatsApp" pré-rempli
  - Galerie photos & vidéos de l'élevage
  - Formulaire de contact avec notification email
  - Back-office admin (CRUD koï, gestion des messages)

  ## Stack

  | Couche | Technologie |
  |---|---|
  | Back-end | Ruby on Rails (RESTful, MVC) |
  | Front-end | Hotwire — Turbo + Stimulus |
  | CSS | Bootstrap / Tailwind CSS |
  | Base de données | PostgreSQL |
  | Authentification | Devise (rôles `visitor` / `admin`) |
  | Upload images | Cloudinary |
  | Emails | ActionMailer |
  | Hébergement | Heroku |
