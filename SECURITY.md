# Security Policy

## Supported versions

The application has not been released yet. Security fixes currently apply to the active development state of the repository.

## Reporting a vulnerability

Please do not open a public issue for security reports.

Share the report privately with the project maintainers and include:

- a short description of the issue
- reproduction steps
- potential impact
- suggested mitigation if available

The team should acknowledge the report, assess severity, and prepare a fix before public disclosure.
